WLPP to MIPS compiler

WLPP is a strict subset of C++. The compiler is written in C++ and it takes takes a tokenized WLPP file, parse it using LR context free grammar, run context sensivtive analysis, and generates equivalent MIPS code.

Demonstrates understanding of main ideas in parsing; including tokenizing with DFA, context free parsing, traversing parse trees, and code generation.

Trie implementation

A C++ program that performs autocompletion. It will maintain a dictionary of available words, and use them to autocomplete prefixes that you provide. Uses standard input, I/O streaming, file streaming to maintain dictionary and input commands.

Accepted commands are:

  • + word
    adds word to the dictionary
  • - word
    remove word to the dictionary
  • ? word
    prints a list of possible autocompletions for word, in alphabetical order
  • include filename
    Reads the file filename and executes the commands contained therein

    pyramid data structure

    A heap based data structure that has two properties:

  • a pyramid P consists of l levels. The ith level, for 0 . i < l, contains at most i + 1 entries, indicated as P(i,j) for 0 <=j <=i. All levels but the last are completely filled, and the last level is left-justified.
  • Any node P(i,j) has at most two children : P(i+1,j) and P(i+1,j +1), if those nodes exist. The priority of a node is always greater than or equal to the priority of either child node
